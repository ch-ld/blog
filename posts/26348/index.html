Linux 命令基础

  • linux

在日常的工作中，或多或少的会接触一些Linux系统，这里简单系统的讲解一下Linux命令。

Linux的哲学思想

  1. 一切皆文件(把几乎所有资源都抽象为文件形式：硬件、通信接口等等)

  2. 由众多功能单一的程序组成

  3. 尽量避免与用户交互(易于以编程的方式实现自动化任务)

  4. 使用纯文本文件保持程序的配置信息

常用命令

cd


change directory，切换目录

ls


list,列出指定目录下的内容

cat


concatinate，连接并倾倒至输出设备

echo


回显命令

mkdir


make directory，创建目录

mv


移动文件或重命名文件

touch


修改文件的时间戳或创建空文件

rm


删除文件

注意：请不要执行“rm -rf /”或“rm -rf/*” ！！！ 谨慎使用

那么命令太多，记不住怎么办呢？没关系，有下列几种方式供你选择。

如何了解一个陌生的命令

使用帮助


使用命令自带的简要格式的使用帮助

使用手册(manual)

  • 存储位置：/usr/share/man
  • 章节：man1,man2,man3,man4……

info 信息


获取命令的在线文档

文档


很多应用程序会自带帮助文档:/usr/share/doc/APPLICATION_NAME-VERSION

  • README:程序的相关信息
  • INSTALL:程序的安装帮助
  • CHANGES：版本迭代的更新信息

搜索引擎


能用Google的尽量用Google，能用必应的尽量用必应，退而求其次就只能用百度咯。
这里推荐一个查询命令的网站https://explainshell.com/，如果你不知道命令的含义，那么打开这个网站查询一下吧。
